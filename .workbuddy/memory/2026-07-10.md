## 关键发现（实测，非仅读文档）
- 18 个 SKILL.md 中，含真实可运行代码的仅上游海外部分：jobbank/jobdanmark/jobindex/jobnet/linkedin/freehire 共 6 个（各有 cli/src/commands，CI 已测）。
- **国内适配层的 9 个 skill 全部只有 SKILL.md，0 代码文件**：bosszhipin-search、domestic-jobs-search、resume-match、interview-mock、job-alert、referral-outreach、resume-build、salary-negotiate、application-tracker。本质是"去装这个外部工具"的指针。
- 头牌功能（国内搜岗：Boss直聘/智联/51job/猎聘/拉勾）**未打包**，仅指针到 boss-cli / get_jobs，无安装器/封装/测试。而真正打包可跑的是丹麦系 CLI（对国内用户无用）。
- 真正落地的国内价值在"中端生成"：apply-zh / da-zhaohu / 08-resume-zh / 09-da-zhaohu-zh / setup-zh（详细 prompt 工作流）。
- 下游闭环（匹配/构建/模拟面试/提醒/内推/谈薪/追踪）全是"自托管重应用"指针：Reactive-Resume、AuraInterviewer(Java18+MySQL+Redis)、jobsync(Docker)、offercatcher(仅 macOS Apple Mail)。普通求职者不会部署。
- CI 仅测上游海外 CLI + lint/tools，未测国内 prompt 层与"复用"集成 → 绿 CI ≠ 国内可用。
- 文档小瑕疵：README.zh 称 08/09 在 `job-application-assistant/zh/`，实际在父目录，zh/ 子目录为空。
- git 干净，国内工作已提交（最新提交围绕 domestic-aware + lint 守卫）。
