# 打招呼话术 / 求职信 示例

> 以下为虚构示例，仅用于演示结构与调性差异。实际生成须基于真实候选画像。

## 一、Boss直聘 短话术（互联网·后端）

**岗位**：某大厂 后端开发（Go），要求 3–5 年、分布式经验
**话术**：
```
您好，我是5年经验的Go后端，看到贵司岗位要求分布式与高并发，
我刚主导过订单系统重构，将接口 P99 从 800ms 降到 120ms、扛住 3 倍流量，
和岗位高度匹配，方便聊聊吗？
```

## 二、Boss直聘 短话术（应届·产品）

**岗位**：某中厂 产品经理（校招），要求 实习经历、数据分析
**话术**：
```
您好，我是XX大学产品方向应届生，有一段大厂增长实习，
独立跑通 A/B 实验把新用户次日留存提升 12%，
匹配贵司"数据驱动"要求，期待沟通～
```

## 三、智联/51job 正式求职信（国企·财务）

**岗位**：某央企 财务分析岗
**求职信**：
```
尊敬的招聘负责人：

您好！我在智联招聘看到贵单位财务分析岗信息，结合4年国企财务经验，
认为与岗位要求高度契合，特此应聘。

在现单位任职期间，我负责经营分析与预算管控，搭建的月度测算模型
将报表周期由5天缩短至2天；这与JD中"经营分析能力"直接对应。
此外，我持有中级会计职称，熟悉国资监管报表体系。

贵单位在[业务领域]的稳健发展令我印象深刻，期待加入贡献力量。
如需进一步了解，可随时联系[电话]。

此致
敬礼
[姓名]　[电话]　[日期]
```

## 四、猎聘 求职信（中高端·技术总监）

**岗位**：某科技公司 技术总监，年薪制、带团队
**求职信要点差异**：突出"汇报对象（CTO）、管理幅度（20人跨端团队）、
业务影响力（营收从 X 到 Y）"，语气更战略化。

---

## 反例（禁用）

- "你好，我对这个岗位感兴趣，期待回复。"（无信息量）
- 整段粘贴简历（Boss 场景阅读以秒计）
- 同一条话术群发所有公司（不点名）
