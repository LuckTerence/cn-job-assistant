## 为什么需要"打招呼话术"

西方求职以 **Cover Letter（求职信）** 为核心，国内（尤其 Boss直聘）以**"打招呼"**为核心动作：
Boss直聘 是"人才与 Boss 直接聊"，一个高质量的开场白直接决定能否进入聊天气泡。
因此国内适配用**打招呼话术**替代 Cover Letter 作为首要产物，并保留**正式中文求职信**
供智联/51job/猎聘等"正式投递"场景使用。

## 一、Boss直聘 打招呼话术（短平快）

### 结构（3 要素，控制在 40–80 字）
1. **身份 + 最强匹配点**：一句话说清"我是谁、我最对口什么"
2. **具体证据**：用 1 个量化结果或硬技能证明匹配
3. **动作/意愿**：表达已读 JD、可尽快沟通

### 模板
```
您好，我是[身份：X年经验的Y方向]。看到贵司[岗位]要求[核心要求]，
我[最相关经历/量化结果，如：主导过同类项目，将XX提升Y%]，
与岗位高度匹配，方便聊聊吗？
```

### 反例（务必避免）
- "您好，我对这个岗位很感兴趣，期待您的回复"（无信息量，必被忽略）
- 大段粘贴简历（Boss 场景阅读时长以秒计）
- 群发同一句（不点名公司与岗位，显敷衍）

## 二、正式中文求职信（智联 / 51job / 猎聘）

### 结构（1 页 / 300–400 字）
1. 开头：应聘岗位 + 信息来源 + 一句话自我定位
2. 主体：2–3 段，每段对应 JD 一个核心要求，用经历+数据证明
3. 结尾：表达对公司的了解与加入意愿 + 礼貌收尾
4. 落款：姓名 / 电话 / 日期

### 模板
```
尊敬的[公司]招聘负责人：

您好！我在[渠道]看到贵司[岗位]招聘信息，结合[X年]的[方向]经验，
认为与岗位要求高度契合，特此应聘。

在[公司]任职期间，我负责[核心职责]，主导[项目]，实现[量化结果]。
这与贵司 JD 中"[某要求]"直接对应。此外，[补充第二、第三匹配点]。

贵司在[业务/价值观]上的实践令我印象深刻，期待能加入团队贡献力量。
如需进一步了解，可随时联系[电话]。

此致
敬礼
[姓名]
[电话]　[日期]
```

## 生成规则（通用）

- **不造假**：话术与求职信中的所有经历、数据必须真实，缺口诚实表述。
- **点名公司与岗位**：每条话术必须含公司名 + 岗位名，杜绝群发感。
- **对齐 JD 关键词**：从职位描述抽取 1–3 个硬性要求作为话术支点。
- **量化优先**：能用数据就不用形容词（"提升 30%" > "显著提升"）。
- **分平台调性**：
  - Boss直聘：口语、直接、敢亮结果
  - 智联/51job：正式、稳重、结构完整
  - 猎聘：突出职级/管理幅度/业务影响力

## 输出文件

- 打招呼话术：`documents/zh/da-zhaohu_<company>_<role>.md`
- 正式求职信：`documents/zh/cover_<company>_<role>.md`
- 由 `/打招呼 <岗位链接或JD>` 命令触发（见 `.claude/commands/da-zhaohu.md`）

## 与西方 Cover Letter 的关系

原 `06-cover-letter-templates.md`（LaTeX/1 页/PDF）保留用于**海外或英文岗**；
本文件用于**国内中文岗**。两者并列，由命令按市场自动选择。
